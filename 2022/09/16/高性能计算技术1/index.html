并行计算概述

背景

算力需求

  • 数据规模急速增长，需要强大的计算资源
  • 数据分析支持通过建模、simulation 和分析，应用在生命科学或生物信息等的各个领域，涉及到巨大的计算量

提升计算机性能的方式

  • 使用多个 CPU

  • 计算机集群 主要授课内容

  • CPU 提速，如量子计算机、、生物计算机与光计算机等

发展现状

  • HPC 算力排名系统

    • 每年 6 月、 12 月更新一次
    • 单位 PFlop/s 含义为千万亿次
    • 百亿亿次为里程碑
      • 关于性能的评判标准
        1. 理论峰值性能
        2. Linpack 程序测试性能
        3. 在普通程序的测试性能
      • 电量要求低于 40kW

并行计算的含义

  • 同义词

    • 并行计算

    • 高性能计算(High Performance)

    • 超级计算(Super computing)

  • 基本条件

    • 硬件

      两部及以上处理机

    • 并行算法设计

      应用可分解为多个子任务，子任务可并行执行

    • 并行编程环境

  • 主要目标

    • 提高求解速度

    • 扩大问题规模

      如使用更多的 CPU

并行计算和分布式计算

分布式计算 并行计算
强调模块间的协调而不注重性能的提升 节点之间距离较近
视频会议 超算

并行计算体系结构

概述

并行计算网络

性能衡量指标

  • 节点度 Node Degree

    射入或射出一个节点的边数

  • 网络直径 Network Diameter

    max(网络中任意两个节点之间的距离)

  • 对剖宽度 Bisection Width

    min(网络等分需要移除的边数)，表明网络的可靠性

  • 对称

    任意节点看网络都是一致的

  • 对剖带宽 Bisection Bandwidth

  • 网络规模

    节点数量或 CPU 数量

时延 Latency

  1. 发送时延
  2. 传播时延
  3. 处理时延
  4. 排队时延
  5. 接收时延

静态互联网络

处理单元间有着固定连接的一类网络

网络 图片 节点度 直径 对剖宽度 备注
一维线性阵列 2 N 1 最简单的互联方式 ，断掉一条边网络不连通 ,FDDI、SCI
一维线性环 2 N/2 2
二维网孔 4 2(根号 N - 1) N
二叉树 3 2(log2N -1) 1 树根部分有显著拥塞
胖树 2(log2N -1) 随着 N 的增大而增大 树根部分有显著拥塞 ，Infiniband(专门为 RDMA 设计的网络、从硬件级别保证可靠传输)
超立方 3 翰林距 1 使用 N 位二进制编码 ,网络复杂，不够直观

动态互联网络

程序执行过程中发生改变

  • 总线结构

    • 待补充图片
    • 接多个设备，带宽共享
    • 同一时刻只能发送一条指令，只连接响应速度相近的设备
    • 节点度 1
    • Diameter 1
    • Size N
    • 易于实现、可扩展性高

  • 交叉开关 Crossbar

    • 节点度 1
    • Diameter 1
    • Size O(N 平方)
    • 性能好，但网络复杂铺设费用昂贵

  • 多级互联网络

标准实现

  • Myrinet
  • 千兆位以太网

以太网

  • 平衡了成本和性能

RoCE

  • 基于 RDMA(Remote Direct Memory Access)的技术

    直接在内存和内存之间进行数据互传，CPU 不需要参与操作，从而减低时延

  • 基于 TCP/UDP 和以太网

  • 实现无损网络

    若发生丢包将会影响性能

RoCE v2

  • 胖树结构

  • 检测拥塞，自动调整到降速模式或停止发送

Top 500 Interconnect Family

  • 性能和系统排名靠前选手

    • Gigabit Ethernet

    • Infiniband

并行计算体系组成要素

  • 处理器
  • 互联网络
  • 内存

并行计算体系耦合方式

  • 共享内存多处理机体系 multiprocessors

    每个处理器都可以直接访问系统任意内存

  • 分布式内存的多计算机体系 multicomputers

    每个处理器都可以直接访问所在网络系统内部内存

DSM (分布式共享内存, Distributed Share Memory)

  • 物理分布式，逻辑共享

  • 内存访问属于非一致内存访问模型(NUMA)

超算系统结构

  • 如 Blue Gene、Atlas 900

          • 内存

    • 流程图
